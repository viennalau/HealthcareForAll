AI Health Assistant

Welcome to the Health Quiz portion of our AI Health Assistant!

If you feel sick, you can use our health quiz with 20 different symptoms as a way to diagnose your sickness! This quiz support four different sicknesses: COVID-19, the flu, the common cold, and allergies. We have plans to implement suggestions to treat your predicted sickness, but for now, our quiz serves as a simple diagnosis.

How does this quiz work, anyway?

  1. We will ask you a series of questions about your symptoms. If you have the symptom listed, then check the box! Otherwise, you can leave the box unchecked.
  2. After you have answered all of the questions, click the submit button to see your results!
  3. When you click the submit button, your results get compiled into a JSON file. We configured a web server by using Flask, a python library, which hosts our trained AI model.
    • We trained our AI model by using a dataset from Kaggle. Our code to train the AI model, along with the data that we used, can be found at this link:
    • AI Health Quiz Code
  4. The web server will then process the JSON file, which contains binary values based on the checkmark's value, unchecked being a 0 and checked being a 1. These binary values match the values that were used in the spreadsheet of symptom and sickness correlation data.
  5. Once the web server processes the request, the web page will display your predicted sickness!
    • We would love to add an accuracy/confidence metric to this prediction, however the library that we used doesn't support that type of metric.

*Please note that this quiz does not replace a professional doctor, and you should consult your care provider with any questions or concerns that you may have.*
